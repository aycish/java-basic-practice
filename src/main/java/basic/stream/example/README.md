## 문제 정의
```

15세 이상은 100만원, 그 미만은 50만원으로 여행 비용이 책정된다.
고객 세 명이 패키지 여행을 떠난다고 했ㅇ르 때, 비용 계산과 고객 명단 검색등에 대한 연산을 스트림을 활용하여 구현해 보자.

고객 정보(Test Input)

CutomerLee
이름 : 이순신
나이 : 40
비용 : 100

CutomerKim
이름 : 김유신
나이 : 20
비용 : 100

CutomerHong
이름 : 홍길동
나이 : 13
비용 : 50


수행할 연산

1. 고객 명단 출력
2. 여행 총 경비 출력
3. 20세 이상인 사람의 이름 정렬하여 출력

```
